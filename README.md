# EduTrack – Student Retention BI Dashboard

Business Intelligence dashboard and predictive analytics system for student retention monitoring using Python, SQL, and visualization tools.

## Problem Statement

Educational institutions often struggle to identify students at risk of dropping out before it becomes critical. This project helps institutions monitor student performance, detect risk patterns, and support early intervention strategies using analytics and predictive modeling.

---

## Business Objective

* Identify high-risk students
* Monitor attendance and academic performance
* Support data-driven decision making
* Improve student retention through early intervention

---

## Features

* Student risk score prediction
* Attendance and academic analytics
* KPI-based dashboard visualization
* Student segmentation (High / Medium / Low Risk)
* Parent alert tracking system
* Real-time analytics and reporting

---

## Tech Stack

* Python
* SQL
* Pandas
* Scikit-learn
* Chart.js
* HTML/CSS
* JavaScript

---

## Dataset

The system processes multiple datasets including:

* Student profiles
* Attendance records
* Academic performance
* Fee/payment details

---

## Key Analytics

* Attendance trend analysis
* Academic risk scoring
* Retention monitoring
* Student performance insights
* Predictive risk analytics

---

## Business Impact

* Automated identification of at-risk students
* Faster intervention for critical cases
* Improved data visibility for faculty and administrators
* Enhanced monitoring of student retention KPIs
